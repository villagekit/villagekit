Built-in traits

```
trait Eq
  type A

  fn eq
    arg x: A
    arg y: A
    res: Bool

  fn neq
    arg x: A
    arg y: A
    res: Bool
  =>
    not Self.eq(x, y)
```

```
enum Comparison
  case LessThan
  case EqualTo
  case GreaterThan


trait Order
  type A

  trait Eq

  fn compare
    arg x: A
    arg y: A
    res: Comparison
```
